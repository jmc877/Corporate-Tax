Required Texts and Materials

  • We're will emphasize learning corporate tax primarily through problems that we cover together in class after the lectures.
  • Corporate Taxation, Examples and Explanations (5th ed), CT E&E, is the primary text for the class. You have free access through the FLS Library. If you wish to purchase a hard copy, make sure it's the 5th ed.
  • The Code and Regs we'll be using this semester are in this pdf: 2026 Corp Tax Code and Regs, which is current through December 2025. I highly recommend that you print it out (it's around 175 pages) and and keep it next to you to mark up during class and while doing the problems and readings. You are welcome to use a commercial printed version, such as CCH or Foundation Press.
  • Here is the spreadsheet listing the Code and Regs sections that was used to generate the class Code & Regs and the webpage of Sarah Lawsky (Code & Regs) where you can create you own. Note, if the pdf exceeds 450 pages, the software will not generate footers with the applicable section numbers, which can make it difficult to figure where you are.
  • The Code and Regs are available on line at Tax Materials. This is a fantastic free database of fundamental tax materials. On the Supplemental Materials page, I show how to make a bookmark that will take you directly to the appropriate code or regulation section. All relevant tax materials are also available on BLaw, WL, Lexis, or Vital Law.
  • For each topic, take a glance at the problems, then read CT E&E, and any additional readings. Use the slides to focus on the important materials that we'll cover during the lectures.
